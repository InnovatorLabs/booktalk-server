## Installation

```bash
$ npm install
$ npm install -g @nestjs/cli
```

## Running the app

- dev mode
  ```bash
  $ npm run start:dev
  ```
- 컴파일된 js 파일 실행
  ```bash
  $ npm run start:prod
  ```

## ENV Sample

.env.dev & .env.prod

```
MONGODB=mongodb://localhost:27017

PORT=3000
```
